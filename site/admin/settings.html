Settings

Public site presentation options stored in app configuration.

Chart legend opacity

Controls how see-through the floating legend panel is (not the chart itself). Desktop uses viewport width > 767px; mobile uses ≤ 767px. If you only set the legacy single value elsewhere, both devices inherit that until you save overrides here. Stored as a decimal in app_config; public pages read it from GET /site-ui (short CDN cache).

75%
75%

Chart legend text brightness

Multiplies label colours inside the floating legend only. 100% leaves palette as designed; below 100% softens text, above brightens it (clamped server-side). Same desktop/mobile split as opacity.

100%
100%

Chart product limit

Upper bound on how many product series the public chart code will draw for the current window (Home Loans, Savings, Term Deposits). Unlimited removes the cap so all in-range products can render (heavier on large windows). Does not change API payloads, only client rendering.

Rate report ribbon (bands mode)

Used only when a rate report chart is in Ribbon view (Home Loans, Savings, Term Deposits). The chart shows each lender as a min–max rate band plus optional per-product lines after you select a lender. Saving writes one JSON object to app_config.chart_ribbon_style; the API merges missing keys with built-in defaults, clamps out-of-range values, and serves the result on GET /site-ui (about one minute cache). After saving, hard-refresh a public report page if you do not see a change immediately.

Band outline (top and bottom edge of each lender band)

Edge width and opacity apply to the band boundary lines. “Others” rows apply while another lender is highlighted or selected so non-focused banks stay subdued.

Band fill (area between min and max)

Base fill applies before focus. Focused fill applies on hover. Selected fill applies after click and should normally be strongest. Others scale multiplies the base fill for banks that are not in focus.

Mean line (average rate within the band)

Optional centre line through each lender band. Set width to 0 to hide. Separate opacities for focused vs other lenders.

Product lines (after a lender is selected)

Individual product rate traces: stroke width and alpha for hover vs click-selected product. Higher z-order banks draw on top; these lines sit above the band stack.

Focus and stacking

Others grey mix pushes non-focused bank colours toward neutral grey (0% = keep hue, 100% = fully greyed). Active z / Inactive z set ECharts z-order for focused vs other lenders; inactive is forced below active on save if needed.
