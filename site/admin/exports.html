Create one single-file SQL dump of the entire D1 database for restore or replacement. Restorable D1 dump

New jobs now build a single `.sql.gz` restore artifact with schema and data. The public download is one file even when the worker assembles it from multiple stored parts behind the scenes.

Restore flow

Use Analyze restore and Restore this dump to repair the current D1 database in place. For a blank or replacement D1 database, download the dump and run the import script below.

node scripts/import-d1-backup.js --db australianrates_api --input ./australianrates-database-full.sql.gz --remote
